ACME UI + MFA Widget

  • This demo web app simulates a Acme.com securing their end user facing website using CyberArk MFA
  • ACME stores the identity of end users in its own database
  • Sign-up and Profile update powered by CyberArk APIs
  • MFA challenges and set-up MFA served by CyberArk Identity MFA widget
  • The backend is driven by Java spring, while Angular is used for the front-end of the application.

The following APIs are used in this web applicaiton -

Sign up

/CDirectoryService/Signup - Create user in the cloud directory service and auto logs in

Profile Update

/user/UpdateProfile - Update current user information.

Logout

/Security/Logout - Logout current user.
