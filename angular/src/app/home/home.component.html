CyberArk Identity API Demo
Explore CyberArk Identity RESTful API capabilities.

The purpose of this demonstration is to show a simple integration of the CyberArk Identity platform features with your website or web applications.

The CyberArk Identity REST APIs expose all features of the CyberArk Identity platform, including the ability to programmatically execute all functions of the CyberArk Identity User Portal. This site shows you how to use REST APIs to:

  • Integrate registration functionality with a web-based application.
  • Add option for a self-service password reset.
  • Allow users to register and log in with their social profiles.
  • Enable single sign-on for multiple web applications.
  • Require users to pass multi-factor authentication challenges to gain access.
  • Customize features of CyberArk Identity platform.

This site has been programmed as a Single Page Application using Angular and Java. This demo is completely open source and can be used in integration projects or simply to learn. The source code can be found on our GitHub.

To learn more about the CyberArk Identity API's, please visit the CyberArk Identity Developer Portal.
