The login form is designed by Acme based on the “/startauthentiation” and “/advanceauthentication” APIs of CyberArk Identity.

To get started the user should be present in the CyberArk Identity directory and should be assigned to a role and a policy. The policy will dictate the MFA challenges that should be posed to the user. By default the user will be asked for “Username” and “Password”.
Once the user is assigned the corresponding MFA policy, the user should register to the MFA for a sucessful login. Else the user would not be allowed to login.
