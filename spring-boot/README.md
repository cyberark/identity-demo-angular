# Java Spring Boot
## Prerequisite
* Follow [Deployment Guide](https://identity-developer.cyberark.com/docs/sample-app-deployment-guide) to setup the sample app environment.

## Build and Run
* Navigate to spring-boot folder.

  ```console
  cd spring-boot
  ```

### Build the Project
* Run below command.

  ```console
  mvn clean install
  ```

### Run the Spring Boot Application Server
* Run below command.

  ```console
  mvn spring-boot:run
  ```
